Boolean

// Boolean constructor new Boolean(value)

All non-null objects are implicitly true. Examples: 

  // non-null object is true
  var result = new Boolean(new Boolean(false));

  // non-null object is true
  if (new Boolean(false)) { /* executes */ }

  // null object is false
  if (null) { /* does not execute */ }
