isFinite()

If the argument is NaN, positive infinity, or negative infinity, this method returns false.

isNaN()

If the argument, when coerced to a numeric value, is an IEEE-794 'Not A Number' value, this method returns true.

encodeURI()

Encodes the entire URI, preserving special chars required for GET and POST requests (e.g., "&" and "="). To encode user entered data, that could contain these chars, use encodeURIComponent() on each component of user data making up the URI.

encodeURI(uri)
